Text to Speech Converter Documentation
----------------------------------------------------------------------------------------------------------------------------
Description:
This Python application allows users to convert text into speech using the Text-to-Speech (TTS) functionality provided by the pyttsx3 library. The user interface (UI) is built using the Tkinter library.
----------------------------------------------------------------------------------------------------------------------------Features:

Text-to-Speech Conversion: Users can enter text into the provided text entry field and click the "Convert" button to generate speech output.
Customizable Properties: The application allows users to adjust the speech rate and volume through customizable properties.
Random Gradient Background: The UI features a visually appealing random gradient background that fills the canvas area.
Stylish UI Components: The labels, text entry field, and convert button are styled using the ttk module to enhance the overall appearance of the application.
--------------------------------------------------------------------------------------------------------------------------
Dependencies:

Python 3.x
Tkinter
pyttsx3
--------------------------------------------------------------------------------------------------------------------------
Instructions:

Ensure Python and the required libraries (Tkinter, pyttsx3) are installed on your system.
Run the script using a Python interpreter.
Enter the desired text into the text entry field.
Click the "Convert" button to hear the text converted into speech
